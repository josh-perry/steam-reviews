# steam reviews
A daily (*dle) game about guessing which Steam games have higher reviews.

## Development
### Quick Start
```bash
npm run install:all
npm run dev
```

## API Endpoints
- `GET /api/health` - Health check endpoint
- `GET /api/games` - Get list of games with ratings for today

## Technologies
### Frontend
- TypeScript
- Lit (Web components)
- Redux
- Webpack

### API
- Node.js
- Express
- TypeScript
- CORS
